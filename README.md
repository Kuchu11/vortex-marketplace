# Vortex Marketplace — Economia Circular Universitária

Projeto desenvolvido para o Desafio Técnico do **Laboratório de Inovação Vortex (UNIFOR)**. A aplicação é uma plataforma web responsiva e instalável (PWA) focada no desapego de materiais acadêmicos (livros, componentes eletrônicos, móveis) entre estudantes.

---

## 🛠️ Tecnologias Utilizadas

### **Backend**
* **Python 3** + **FastAPI**: Framework assíncrono para criação da API RESTful.
* **SQLite** + **SQLAlchemy**: Persistência de dados e mapeamento relacional.
* **Uvicorn**: Servidor ASGI para execução da API.

### **Frontend & PWA**
* **HTML5, CSS3 e JavaScript Puro (Vanilla JS)**: Interface responsiva, moderna e otimizada.
* **Progressive Web App (PWA)**: Manifesto web (`manifest.json`) e Service Worker (`sw.js`) para suporte a instalação e armazenamento em cache.
